# VIT Project Management Dashboard

A lightweight, frontend-based invoice generator system built for VIT (Venture IT).  
It allows users to quickly create professional, print-ready invoices with automatic calculations and PDF export.

---

## 🚀 Features

- Create professional invoices instantly
- Add multiple service/product items
- Auto-calculation (Subtotal, Received Amount, Due Amount)
- Fixed company branding (logo, header, signature, stamp)
- Print-ready A4 layout design
- Download invoice as PDF
- Fully responsive (mobile-first design)
- Works offline (no backend required)

---

## 📦 Tech Stack

- HTML
- CSS
- JavaScript
- jsPDF
- html2canvas

## 🧾 Invoice System

Each invoice includes:

- Company header (fixed)
- Client details (dynamic)
- Project/service items table
- Financial summary:
  - Subtotal
  - Received Amount
  - Due Amount
- Footer branding:
  - Payment Received stamp
  - Digital signature

---

## 📄 Output

- Print directly from browser
- Download as high-quality PDF
- Share with clients instantly

---

## 📱 Design

- Mobile-first responsive UI
- Clean and minimal UI for fast usage
- Optimized for low-end devices

---

## ⚙️ Usage

1. Open `index.html`
2. Click **Create Invoice**
3. Fill in client and project details
4. Generate invoice
5. Download or print PDF

---

## 📌 Purpose

This system is built to simplify invoice creation for VIT by removing manual document design and enabling fast, consistent, and professional billing.
